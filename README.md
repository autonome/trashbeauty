# TrashBeauty

### Sorting images from in-home connected cameras.

*By: Davide Gomba, Dietrich Ayala, Ame Elliot and Rikta Krishnaswamy at the Berlin Design Sprint, April 2016.*

VALUE PROPOSITION  Better Treasure:Higher quality photos thanks to ambient capture.Better Trash: Improved data removal experience providing empowerment and user control of data.USER NEEDFamily photos are valuable, but busy families can’t always document the birth and childhood of their children – especially the second child.
